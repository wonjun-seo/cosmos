Canvassing

Dataset: Door-to-door canvassing broockman_kalla_replication_data.dta

Source: http://dx.doi.org/10.7910/DVN/WKR39N

Original article: David Broockman, Joshua Kalla ,Durably reducing transphobia: A field experiment on door-to-door canvassing.Science352,220-224(2016).DOI:10.1126/science.aad9713

This starter notebook helps you load, inspect, and perform basic exploratory analysis on the broockman_kalla_replication_data.dta file from Broockman & Kalla (2016). Some codes are adapted from the original authors' Broockman-Kalla-SM.RMD. You can download the processed file used in this notebook via this link.

Description for the variables can be found in Table S24 in Broockman-Kalla-SM.pdf.

Basic cleaning of missing data.

In[3]:
# Impute missing age
df['vf_age'] = df['vf_age'].fillna(df['vf_age'].mean())

# Fix Spanish interview indicator
if 'survey_language_es' in df.columns and 'survey_language_t0' in df.columns:
    mask = df['survey_language_es'].isna()
    df.loc[mask, 'survey_language_es'] = (df.loc[mask, 'survey_language_t0'] == 'ES').astype(float)
    df['survey_language_es'] = df['survey_language_es'].fillna(df['survey_language_es'].mean())

# Keep only contacted voters (door reached)
df_contact = df[df['contacted'] == 1].copy()
print(df_contact.shape)
